XAMPP adalah software gratisan dan open source yang dapat diinstall di berbagai platform, seperti Windows, Linux, maupun OS X1234. XAMPP adalah sebuah web server full package atau standalone, yang mana dapat berdiri sendiri. Di dalam aplikasi XAMPP terdapat puluhan paket modul, bahasa pemrograman, hingga komponen lain. XAMPP digunakan sebagai local web server untuk pengujian aplikasi web melalui localhost.
